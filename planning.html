Display.

Esta propiedad puede tomar muchos valores diferentes, pero los mas comunes son estos tres:

* block

Hace que el comportamiento del elemento sea como un bloque.

* inline

El elemento se renderizara en linea con otros elemento.

* inline-block

El elemento tendra un comportamiento mezcla entre los dos anteriores.

Diferencias entre Block, inline, inline-Block

Block (Display:Block;)

Elementos de bloque.

Ciertas etiquetas se renderizan en el navegador en lineas independientes, no mezcladas con el reto del texto. Ejemplos de estas etiquetas son los encabezados (< h1 > hasta ), las citas en bloque (< blockquote >), por supuesto los parrafos (< p >), y quiza la mas conocida de todas que es la es la etiqueta (< div >) usada normalmente para envolver a otros elementos.
Estos elementos se les denomina (elementos de bloque).

Iniline (Display:inline;)

Elementos en linea.

Para explicarlo de manera sencilla, y simplificando, ciertas etiquetas HTML no afectan en absoluto al flujo de los demas elementos, limitandose a marcar ciertos fragmentos con una determinada semantica y dejando que el texto y otros elementos contiguos sigan fluyendo en la misma linea, colocandose a ambos lados de dicho elemento.
Ejemplos de estas etiquetas son: (< a >, < em >, < del > ...) y quiza la mas util de todas: (< span >), que se usa para envolver elementos en-linea "inline"

Inline-block (Display:inline-block;)

Elementos conbinados en linea + bloque.

Bien, aparte de este comportamiento por defecto, es posible cambiar el mmodo en el que se visualiza y/o renderiza cualquier elemento HTML de una pagina usando la propiedad CSS llamada display.
